# Stitch & Spec

Sewing and embroidery machine research site for Evolen.

## Stack

- Astro static site
- Cloudflare Pages target
- Sitemap generated by `@astrojs/sitemap`
- IndexNow helper script in `scripts/indexnow.mjs`

## Local commands

```bash
npm install
npm run build
npm run preview
```

## Cloudflare Pages build settings

- Framework preset: Astro
- Build command: `npm run build`
- Build output directory: `dist`
- Environment variable: `PUBLIC_SITE_URL=https://stitchandspec.com`
